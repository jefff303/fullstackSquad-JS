Gérer les Attributs HTML en JavaScript

La manipulation des attributs HTML, des classes CSS et des styles est essentielle pour créer des interfaces utilisateur dynamiques et interactives.

👉 Pourquoi manipuler les attributs ?

  • Modifier dynamiquement le contenu d'une page (ex: changer une image via l'attribut `src`).
  • Créer des états visuels (ex: désactiver un bouton via l'attribut `disabled`).
  • Stocker des données personnalisées (ex: `data-id`) pour un usage en JavaScript.

⚡ Gérer les attributs : `setAttribute` & `getAttribute`

Ces deux méthodes sont la manière la plus courante de manipuler les attributs. 

                
// Supposons un bouton dans le HTML : 
const bouton = document.getElementById('mon-bouton');

// Définir un attribut
bouton.setAttribute('disabled', '');
// Le HTML devient : 

// Lire un attribut
const isDisabled = bouton.getAttribute('disabled');
console.log(isDisabled); // '' (ou 'true' selon le navigateur)

// Supprimer un attribut
bouton.removeAttribute('disabled');

🌱 Gérer les classes : `classList`

La propriété `classList` est la manière moderne de manipuler les classes CSS d'un élément, offrant des méthodes simples pour ajouter, supprimer ou vérifier la présence d'une classe. 

                
const card = document.querySelector('.card');

// Ajouter une classe
card.classList.add('bg-blue-500');

// Supprimer une classe
card.classList.remove('bg-base-100');

// Basculer une classe (l'ajoute si elle n'existe pas, la supprime si elle existe)
card.classList.toggle('shadow-xl');

// Vérifier si une classe est présente
if (card.classList.contains('bg-blue-500')) {
  console.log('La carte est bleue.');
}

✅ Gérer les styles : `style`

La propriété `style` permet d'accéder et de modifier les styles en ligne (inline) d'un élément.

  • Les noms de propriétés CSS avec tirets (ex: `background-color`) deviennent des noms en camelCase (ex: `backgroundColor`).
                
const titre = document.querySelector('h1');

// Modifier un style
titre.style.color = 'red';
titre.style.fontSize = '3rem';

// Lire un style
const couleurTitre = titre.style.color;
console.log(couleurTitre); // 'red'
