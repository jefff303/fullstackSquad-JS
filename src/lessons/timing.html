⏱️ Gérer le Temps en JavaScript

Apprenez à utiliser les fonctions de **timing** du navigateur pour exécuter du code après un délai (`setTimeout`) ou de manière répétée (`setInterval`). Ces fonctions sont au cœur de l'asynchronisme côté client.

1. Exécution unique avec setTimeout()

`setTimeout()` exécute une fonction **une seule fois** après un délai spécifié (en millisecondes). Il est parfait pour différer une action (par exemple, afficher une notification d'erreur après 3 secondes).

Syntaxe :


const timerId = setTimeout(fonctionAExecuter, delaiEnMillisecondes);

Exemple :


const message = 'Ce message apparaît après 2 secondes.';

// La fonction s'exécute après 2000 millisecondes
const timeoutID = setTimeout(() => {
    console.log(message); 
}, 2000);
**Annulation :** Utilisez clearTimeout(timerId) pour annuler l'exécution avant qu'elle ne se produise.

2. Exécution répétée avec setInterval()

`setInterval()` exécute une fonction de manière **répétée** à chaque intervalle de temps spécifié. C'est la fonction utilisée pour créer des horloges ou des carrousels automatiques.

Syntaxe :


const intervalId = setInterval(fonctionAExecuter, intervalleEnMillisecondes);

Exemple (Horloge Simple) :


let counter = 0;

// La fonction s'exécute toutes les 1000 millisecondes (1 seconde)
const intervalID = setInterval(() => {
    counter++;
    console.log(`Le temps passe... ${counter}s`);
    
    if (counter >= 5) {
        // Obligatoire : arrêter l'intervalle après un certain temps
        clearInterval(intervalID);
        console.log("Intervalle arrêté.");
    }
}, 1000);
**Arrêt crucial :** Vous DEVEZ utiliser clearInterval(intervalId). Sans cela, l'exécution se poursuivra indéfiniment, consommant des ressources.

🧠 Concept Asynchrone

Les fonctions de timing ne bloquent pas le fil d'exécution principal de JavaScript. Elles sont envoyées à la **Web API** du navigateur, qui les gère en parallèle et les renvoie à la file d'attente d'événements (Event Loop) une fois le délai écoulé.

🚀 Exercice pratique

Créez un élément **div** dans votre page. Utilisez `setInterval()` pour changer sa couleur de fond (`element.style.backgroundColor`) toutes les 500 millisecondes. Arrêtez le changement après 10 secondes en utilisant `setTimeout()`.
