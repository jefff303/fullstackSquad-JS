Les librairies en JavaScript

Découvrez comment utiliser des librairies dans vos projets, que ce soit avec un lien CDN ou via npm.

Qu’est-ce qu’une librairie ?

Une librairie est un ensemble de fonctions ou de composants prêts à l’emploi, qui vous aide à coder plus rapidement. Exemple : jQuery, Lodash, Chart.js, etc.

Utilisation via CDN

Le CDN (Content Delivery Network) permet de charger une librairie directement depuis un serveur externe, en ajoutant une balise <script>. 



<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

<script>
  $(document).ready(function(){
    console.log("jQuery est prêt !");
  });
</script>

✅ Avantage : rapide à mettre en place. ❌ Inconvénient : dépend d’internet et de la disponibilité du serveur CDN.

Utilisation avec npm

Avec un gestionnaire de paquets comme npm ou pnpm, vous pouvez installer vos librairies dans votre projet. 


# Installation de Lodash
npm install lodash

# Ou avec pnpm
pnpm add lodash

Ensuite vous pouvez l’utiliser dans votre code : 


import _ from "lodash";

const numbers = [1, 2, 3, 4, 5];
console.log(_.shuffle(numbers)); 
// Exemple : mélange le tableau

Exemples de librairies utiles

  • Axios : pour faire des requêtes HTTP plus simples que fetch.
  • Day.js : pour manipuler les dates facilement.
  • Chart.js : pour créer des graphiques interactifs.
  • Lodash : utilitaires pour les tableaux, objets, chaînes de caractères.

🚀 Exercice pratique

Installez dayjs avec npm ou pnpm, puis affichez la date et l’heure actuelles formatées dans la console.
